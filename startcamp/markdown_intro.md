# 리스트

### 순서가 있는 리스트

1. `1.` 리스트가 만들어 진다
2. 엔터치면 자동으로 다음 숫자가 만들어 진다.
   1. `Tab`누르면 세부 사항도 만들 수 있다.
3. `Shift + Tab`누르면 상위 수준으로 나갈 수 있다.
4. 완전히 나가고자 한다면 가장 박에 있는 리스트에서 한 번 더 `Shift + Tab`



### 순서가 없는 리스트

-  `- `

- 얘도 순서가 잇는 리스트와 완전히 같다

  - `Tab`하면 들여쓰기

    1. 순서가 있는 리스트 / 없는 리스트 섞어 쓸 수 있다.

## 코드와 관련된 기능

### code block

`` <- 코드 블럭

`그냥 색만 넣어주는 기능 아님.`

문장을 쓰던 도중에 `print()`라는 코드를 써야 한다면, 코드의 가독성을 늘리기 위해 사용한다.





### Code snippet

> 백틱 ` 3개를 한번에
>
> 그 뒤에 쓰고자 하는 언어

```python
print('hello World!')
```

Code snippet은 같은 방식으로 여러 사이트에서 사용할 수 있다.
