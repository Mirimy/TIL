## CLI INTRO

### 각종 명령어들

---

#### 파일 목록 확인

``` bash
$ ls
```

``` bash
$ ls -a
```

./ 현재폴더

../ 상위폴더

``` bash
$ ls -l
```

파일 상세정보

---

#### 작업 위치

``` bash
$ pwd
```

현재 작업 위치 경로 확인

``` bash
$ cd {폴더명}
```

해당 폴더로 작업 위치 이동

폴더명 입력하다가 `Tab`누르면 자동완성

``` bash
$ cd ..
```

상위 폴더로 이동 (.. 과 폴더명을 복합해서 사용 가능 /ex : cd ../../{폴더명})

---

#### 파일 생성

```bash
$ touch {파일명.확장자}
```

`{}`~~는 가변형~~

#### 폴더 생성

``` bash
$ mkdir {폴더명}
```

make directory

---

#### 파일 삭제

``` bash
$ rm {파일명.확장자}
```

#### 폴더 삭제

``` bash
$ rm -r {폴더명}
```
